## 🎯 Problema Focado

O principal problema abordado foi a dificuldade dos torcedores em enfrentar filas para comprar comida e a perda de momentos importantes do jogo.

---

## 💡 Justificativa de Design

O protótipo foi desenvolvido com foco na usabilidade em ambientes com alto nível de distração, como estádios.

Foram utilizadas as seguintes estratégias:

* Botões grandes para facilitar o clique
* Informações principais destacadas (como placar)
* Navegação simples e direta
* Redução de textos longos

---

## 🔄 Fluxo do Usuário

### 🍔 Pedido de Comida:

1. O usuário acessa a tela inicial (Home)
2. Clica em "Lanchonete"
3. Escolhe os itens no cardápio
4. Visualiza o carrinho
5. Finaliza o pedido

---

### 🎥 Visualizar Replay:

1. O usuário acessa a Home
2. Clica em "Replay"
3. Seleciona o lance desejado

---

## 🧠 Considerações de UX

O design foi pensado para funcionar bem mesmo com:

* Barulho
* Pressa
* Distração com o jogo

Priorizando rapidez e clareza nas ações.
